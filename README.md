# ✚ Sunrise Pharmacy — Inventory Management System

A full computerized pharmacy inventory and dispensing system built for **Sunrise Pharmacy, Nairobi, Kenya**. Manages medicines, hospital equipment, patient prescriptions, and financial reporting — all in one system.

---

## 🔐 User Roles

| Role | Access |
|------|--------|
| **Administrator** | Full access — add, edit, delete items, dispense, view reports |
| **Cashier** | Dispense prescriptions, process payments, print receipts |
| **Financial Manager** | View sales history, revenue reports, profit analysis |

---

## 💊 Key Features

- **Medicine & Equipment Inventory** — Add, edit, delete items with Kenyan (KSh) pricing
- **Prescription Dispensing** — Enter patient details, select medicines, auto-deduct stock
- **Receipt Generation** — Printable receipt for every transaction
- **Sales History** — Full transaction log with patient and payment details
- **Financial Reports** — Revenue, profit margin, top-selling items, payment breakdown
- **Stock Alerts** — Automatic low stock and out-of-stock warnings
- **Payment Modes** — Cash, M-PESA, Insurance, Credit Card

---

## 🚀 How to Run

1. Download `index.html`
2. Open it in any web browser — no installation required
3. Log in with your role credentials
